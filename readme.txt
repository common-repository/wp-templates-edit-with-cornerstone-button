Get the "Edit with Cornerstone" button back when you are in a page.

== Description ==
This is a really simple addition to your X Theme site. 

Gutenberg removed the "Edit with Cornerstone" button, so, this adds it back in.

You'll see it at the end of your admin bar when available. 

Works for X and Pro and only on Pages. 

== Installation ==

<h4>Using The WordPress Dashboard</h4>
<ol>
<li>Navigate to Add New from your dashboard</li>
<li>Search for Edit with Cornerstone</li>
<li>Click Install Now</li>
<li>Activate the plugin</li>
</ol>
<h4>Uploading via WordPress Dashboard</h4>
<ol>
<li>Navigate to the Add New in the plugins dashboard</li>
<li>Navigate to the Upload area</li>
<li>Select <code>wptemplates-edit-with-cornerstone.zip</code> from your computer</li>
<li>Click Install Now</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>
<h4>Using FTP</h4>
<ol>
<li>Download <code>wptemplates-edit-with-cornerstone.zip</code></li>
<li>Extract the <code>wptemplates-edit-with-cornerstone</code> directory on your computer</li>
<li>Upload the <code>wptemplates-edit-with-cornerstone</code> directory to the <code>/wp-content/plugins/</code> directory</li>
<li>Activate it from the Plugins dashboard</li>
</ol>

== Frequently Asked Questions ==
<b>Do I need to change any settings?</b>

No. This plugin just works without any need to do anything extra after activating it.
